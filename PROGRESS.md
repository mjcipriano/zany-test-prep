## Notes / decisions

- Content is generated by deterministic Python generators (math computed; RW
  rule-based/combinatorial; authored reading passages) â†’ guaranteed-correct, auditable.
- Persistence is JSON in `shared_preferences` behind a `KeyValueStore` interface
  (easy to fake in tests; swap to Drift/Isar later if needed).
- Widget tests inject the content bundle synchronously and use an in-memory store, so
  the suite is deterministic and headless (no device needed for CI).
